# cantina Escolar - banco de Dados

## Objetivo
Criar e manipular um banco de dados MySQL para uma cantina escolar,
utilizando comandos SQL de criação, inserção, atualização, exclusão e consulta.

## Ferramenta utilizada
MySQL Workbench

## Como executar
1. Abrir o MySQL Workbench
2. Conectar ao servidor MySQL
3. Abrir o arquivo recuperacao.sql
4. Executar o script completo
